## How to Run

### Prerequisites
- Node.js (v14+)
- MongoDB
- npm or yarn

### Quick Start

**Terminal 1 - Backend:**
```bash
cd Backend
npm install
npm start
```

**Terminal 2 - Frontend:**
```bash
cd Frontend
npm install
npm start
```

**Access:**
- Frontend: http://localhost:3000
- Backend: http://localhost:5000
